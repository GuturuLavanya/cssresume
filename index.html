PROJECTS

MULTI DISEASE PREDICTION USING MACHINE LEARNING

Human disease prediction systems are becoming increasingly important for early diagnosis and treatment of diseases.Many of the existing machine learning models for health care analysis are concentrating on one disease per analysis. Like one analysis for diabetes analysis, one for cancer analysis, one for skin diseases like that. There is no common system where one analysis can perform more than one disease prediction. Our diagnosis system is able to predict more than one analysis. The system is trained on a datasets of 149 symptoms and 58 diseases. A correlation heat map is used to analyze the relationship between symptoms, and a random forest classifier is used to predict diseases from symptoms. The system achieves an accuracy of 99% on the training datasets. The system has a number of applications including helping doctors, educating patients and helping in developing diagnostic tools. The system is a valuable tool for early diagnosis of diseases, and it has the potential to improve the quality of healthcare for millions of people.
